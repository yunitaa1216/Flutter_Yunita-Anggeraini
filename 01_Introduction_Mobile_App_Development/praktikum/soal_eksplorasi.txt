1. Jelaskan perbedaan antara pengembangan aplikasi mobile secara hybrid (contoh: Flutter) dan pengembangan aplikasi mobile secara native (contoh: Kotlin untuk pengembangan aplikasi Android).
Jawaban:
Hybrid adalah cara pengembangan aplikasi menggunakan satu basis kode untuk diimplementasikan pada lebih dari satu platform (android dan iOS). Pengembangan aplikasi secara hybrid memiliki kelebihan yaitu pengembangannya lebih cepat dan efisien dan biaya pengembangannya lebih rendah daripada 
aplikasi native karena tidak memerlukan banyak waktu dan sumber daya. Selain kelebihan tersebut, aplikasi hybrid juga memiliki kekurangan seperti akses ke fitur dan fungsi khusus platform terbatas tidak mudah diakses dengan menggunakan teknologi berbasis web sehingga membatasi kemampuan aplikasi dan membuat aplikasi menjadi kurang efisien dibandingkan aplikasi native.

Native merupakan aplikasi atau program yang dibuat dalam bahasa pemrograman tertentu untuk platform tertentu. Misalnya untuk platform Android bahasa 
pemrograman Java atau Kotlin. Kelebihan pengembangan aplikasi secara native yaitu aplikasi dapat dikembangkan menjadi sangat optimal dan memiliki kinerja lebih baik, serta memiliki akses penuh ke platform atau perangkat yang digunakan misalnya kamera, map dll. Selain kelebihan, aplikasi native juga memiliki kekurangan yaitu membutuhkan lebih banyak waktu dan sumber daya dalam pengembangannya, dan hanya dapat berjalan di platform yang cocok atau sesuai dengan bahasa pemrograman yang digunakan.
