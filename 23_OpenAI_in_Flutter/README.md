<h1><center>OpenAI in Flutter</center></h1>

<p>OpenAI adalah sebuah library yang digunakan untuk memanfaatkan teknologi AI(Artificial Intellegence) yang dapat di integrasikan dengan berbagai aplikasi seperti API, web dan mobile.</p>

<p>Aplikasi berbasis AI misalnya face recognition, speech recognition, autonomous vehicle, dan sentiment analysis.</p>

### Keuntungan dari OpenAI
- Gratis
- Mudah dipasang
- Dapat digunakan di berbagai perangkat
- Jumlah parameter: 175 Miliar(model davinci 03)
